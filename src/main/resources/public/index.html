This system processes messages containing the following data (in JSON format):


  • Sales Type
  • Sales Amount
  • Sales Quantity
  • Sales Operations
Sample request:

{ "type":"apple", "quantity" : 20, "amount":"10p", "operation":"" }

Assumptions made in this project:

1. Sales amount is always in pence (Quantity x Amount will be recorded in £(Pounds))
2. If Sales operation is set, adjustments are to be made on existing record - if current sale is 0, no adjustments will be made.
3. If operation is not one of the expected operations, no adjustments will be done.
Expected operations are the following:
  • ADD
  • SUBTRACT
  • MULTIPLY
